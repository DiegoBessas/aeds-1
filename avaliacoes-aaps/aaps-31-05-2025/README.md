# AAPS — 31/05/2025 — Bibliotecas Python Mais Utilizadas Atualmente

## Contexto

Avaliação prática aplicada próximo ao fim do semestre, com um formato diferente das demais: em vez de exercícios de lógica a resolver, o material é um **material de pesquisa** listando 10 bibliotecas populares do ecossistema Python — cada uma com uma breve explicação e um trecho de código de exemplo — cobrindo computação numérica, ciência de dados, machine learning/deep learning e desenvolvimento web.

Bibliotecas apresentadas: **NumPy**, **Pandas**, **Matplotlib**, **Scikit-Learn**, **TensorFlow**, **Keras**, **Flask**, **Django**, **Requests** e **FastAPI**.

**Enunciado da atividade:** "Com base em cada item anterior, faça uma pesquisa e crie um código simples, mas funcional para cada item."

Ver o material completo em [`bibliotecas-python-mais-utilizadas.pdf`](./bibliotecas-python-mais-utilizadas.pdf).

> **Observação:** o material enviado contém apenas o PDF de referência da atividade; não havia, junto a ele, arquivos `.py` com a resolução (os códigos autorais pedidos pelo enunciado, um para cada uma das 10 bibliotecas). Caso você ainda tenha essa resolução em algum outro lugar, posso incluí-la aqui.
